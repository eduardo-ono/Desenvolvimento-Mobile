# MVC (Model-View-Controller)

Prof. Eduardo Ono

<br>

## Conceitos

> O __*MVC*__ (_Model-View-Controller_) é um padrão de arquitetura de software que divide uma aplicação em três camadas: _Model_, _View_ e _Controller_.
